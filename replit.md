# Dr. Doshie Piper - Personal Website

## Overview
A static HTML/CSS personal portfolio website for Rev. Dr. Doshie Piper - Criminologist, Speaker, and Justice Advocate.

## Recent Changes
- **Jan 2026**: Synchronized Projects page architecture with full dark indigo gradient theme, glassmorphism cards, and premium styling throughout
- **Jan 2026**: Added high-end "Strategic Consulting & Grant Development" section to Projects page with premium service card design
- **Jan 2026**: Added "2026 Engagement Highlights" section to Media page with horizontal card layout featuring MLK Youth Summit, Ella Austin Legacy Award, and DreamWeek Speaker images
- **Jan 2026**: Added "Latest News" badge to Home page announcing MLK Commission Chairperson role

## Project Structure
- `index.html` - Homepage with latest news badge
- `about.html` - About page
- `projects.html` - Projects page
- `publications.html` - Publications page
- `media.html` - Media page with 2026 Engagement Highlights section
- `contact.html` - Contact page
- `style.css` - Main stylesheet
- `images/` - Image assets
- `docs/` - Document files (CV)
- `server.py` - Python HTTP server with clean URL support

## Running the Project
The website runs on a Python HTTP server at port 5000. The server handles clean URLs (e.g., `/about` serves `about.html`).

## Deployment
Configured for static deployment serving all HTML, CSS, images, and documents from the root directory.
